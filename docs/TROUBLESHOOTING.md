# ADDB Sovereign Architect v1.0.0 - Troubleshooting Guide

## Common Error Codes
| Code | Description | Resolution |
|------|-------------|------------|
| ERR_WS_DISCONNECT | WebSocket connection lost | Check backend server status |
| EVO_BUILD_FAIL | Evolution patch failed to build | Review lint logs in SYSTEM EVOLUTION |
| SCANNER_CRITICAL_ALERT | Security leak detected | Immediate lockdown of affected modules |
